Markdown的一些使用技巧

发表于 2016-03-01   |   分类于 技术   |     |  

    这是在使用Markdown时遇到的一些问题，其实不过是在md里使用了HTML的语法去实现罢了。不过大神们都觉得这样使用md的话，就失去md的写作意义了。不论如何，权当码来一乐吧

插入空格

半角空格：&ensp;或 &#8194;
全角空格：&emsp; 或 &#8195;
(注意分号不能丢。)
例：     111

首行缩进

&nbsp;
例：     缩X5

修改字号

  • <span style="font-size: 32px;">我是变大的字</span>
    例：我是变大的字

  • <font size=10>我是缩小的字</font>
    例：我是缩小的字

修改文字颜色

<font color=#B22222>我是姨妈红</font>

例: 我是基佬紫.    我是姨妈红.
